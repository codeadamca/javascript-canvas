Sizing the Canvas

When including a canvas into an HTML document you need to consider two sets of widths and heights. First, there is the width and height that was determined when saving the image from Photoshop or Illustrator. Second, you determine the size the image will appear using styles.

Likewise with a canvas, you have two sets of dimensions. First, are the width and height set in the canvas tag (or with JavaScript), these are like the dimensions set in Photoshop. Second, you have the dimensions set in your styles, the actual visual size of the canvas.

Just like an image, do not set your canvas to 200 by 200 pixels and then display it 600 by 600 pixels, or it will appear blurry.
